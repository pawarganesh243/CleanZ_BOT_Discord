# Privacy Policy for CleanZ Bot

**Last Updated:** July 13, 2026

Thank you for using CleanZ Bot! We take your privacy seriously. This document outlines how we handle data when you use the bot.

### 1. Data Collection
CleanZ Bot **does not** collect, store, or process any personal user data, message content, or server information. 

### 2. How the Bot Works
When the `/clean` command is used, the bot temporarily reads message timestamps in the current channel's history to determine which messages to delete. This data is held entirely in memory and is instantly discarded once the deletion process is complete. 

### 3. Databases and Logging
- We do not use any databases.
- We do not log message content.
- We do not track user activity.

### 4. Contact
If you have any questions or concerns about this Privacy Policy, please reach out to the bot developer.
